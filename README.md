# 커뮤니티 게시판 

## 만든 목적
- 기본이 되는 게시판 기능과 그에 따른 파일, 댓글, 페이징, 로그인(security 사용) 기능 처음부터 스스로 구현하기 위해 제작
- 참여도 : 개발 100%

# 사용 기술 및 개발 환경
- O/S : window10
- Language : JAVA17
- Framework : Spring Boot 3.2.2
- SQL : MariaDB
- Tool : IntelliJ, Git(SourceTree), HeidiSQL
